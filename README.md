# 英単語アプリ

英単語を学習するためのフラッシュカード形式のアプリです。

## 機能

- **単語テスト機能**: フラッシュカード形式で単語と意味を表示
- **OK/NG判定**: 自分でボタンを押して判定
- **未判定単語のみ表示**: NGか判定していない単語だけが出てくる
- **進捗機能**: 何%進んでいるかわかるように表示
- **音声機能**: 単語や意味を音声で聞ける
- **リセット機能**: 進捗をリセットして最初から学習可能
- **レスポンシブデザイン**: スマホにも対応した優しい色合いのUI

## 技術スタック

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- localStorage (データ保存)

## セットアップ

1. 依存関係をインストール:
```bash
npm install
```

2. 開発サーバーを起動:
```bash
npm run dev
```

3. ブラウザで [http://localhost:3000](http://localhost:3000) を開く

## 使い方

1. アプリを開くと、単語が表示されます
2. 「意味を表示」ボタンを押して意味を確認します
3. 理解できた場合は「✅ OK」、できなかった場合は「❌ NG」を押します
4. NGか判定していない単語だけが次々と表示されます
5. 進捗バーで学習の進捗を確認できます
6. 🔊 ボタンで単語や意味を音声で聞くことができます
7. 進捗をリセットしたい場合は「🔄 進捗をリセット」ボタンを押します

## データ

単語データは `data.md` に保存されています。このファイルを編集することで、学習する単語を変更できます。

## ビルド

本番環境用にビルドする場合:

```bash
npm run build
npm start
```
